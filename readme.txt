== Description ==
Classic Social Share is a simple plugin that adds social media sharing buttons to your WordPress posts and pages.

== Installation ==
1. Upload the `classic-social-share` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings in the 'Settings' -> 'Classic Social Share' menu.

== Frequently Asked Questions ==
= How do I add the social share buttons to my posts? =
The buttons will automatically appear at the bottom of your posts and pages once the plugin is activated.

= Can I customize the appearance of the buttons? =
Yes, you can customize the appearance of the buttons in the plugin settings.
